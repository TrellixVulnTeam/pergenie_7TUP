  • What is VCF?

    VCF (Variant Call Format) is a tipical data format for DNA sequencing data from NGS.

    Example:

    ##fileformat=VCFv4.1
...
#CHROM POS     ID        REF    ALT     QUAL FILTER INFO                FORMAT      NA00001

  • What is VCF (Whole Genome) or VCF (TruSeq Exome)?

    VCF file for Whole Genome contains whole (all) the genomic positions of your genome.

    VCF file for TruSeq Exome contains targetted region of TruSeq Exome Enrichment Kit (Illumina, Inc.) of your genome.

  • What is 23andMe?
